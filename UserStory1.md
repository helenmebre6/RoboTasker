# User Stories

## 1. As a user, I want to be able to select my recipe with one click, so that I have an efficient execution process

![alt text](https://user-images.githubusercontent.com/127349077/226676828-d2e2c550-2082-4bc6-8ac8-3da1061fa562.jpg)

## 2. As a user, I want to be able to choose from a list of pre-installed recipes, so that I can begin my dinners more quickly
![alt text](https://user-images.githubusercontent.com/127349077/226676764-2d0bee30-fc55-4f67-8948-31283338c82d.jpg)

## 3. As a user, I want to be able to easily search for pre-installed recipes, so that I can quickly find the recipe I want to make without scrolling through a long list
![alt text](https://user-images.githubusercontent.com/127349077/226676777-6f3868e2-8314-49d6-b87f-e7c8927d1737.jpg)

## 4. As a user, I want to be able to customize my recipes, so that I am enjoying my meals and adding a personal touch
![alt text](https://user-images.githubusercontent.com/127349077/226676786-08022f21-7585-490f-862a-203bffe0d512.jpg)

## 5. As a user, I want to be able to customize my profile on the app and add recipes into categories
![alt text](https://user-images.githubusercontent.com/127349077/226676788-fde10b4f-eee4-419c-9f2e-9dc8397fd34b.jpg)

## 6. As a user, I want to be able to input custom parameters for each recipe, so that the RT5000 can perform the task according to my preferences
![alt text](https://user-images.githubusercontent.com/127349077/226676791-eed4a015-2df8-4677-96e6-e1e1f8502587.jpg)

## 7. As a user, I want to be able to schedule the RT5000 to start the task at a specific time, so that the task is completed when I need it to be
![alt text](https://user-images.githubusercontent.com/127349077/226676796-f79c2663-ff3c-4b4d-a4f2-fb95cc566b54.jpg)

## 8. As a user, I want the RT5000 to provide me with audio and visual feedback during the task, so that I know that it is working correctly
![alt text](https://user-images.githubusercontent.com/127349077/226676805-d0dd18ee-b529-4d62-92f4-a1f83473e0d6.jpg)

## 9. As a user, I want the RT5000 to be able to handle multiple recipes in a single session, so that I can complete multiple tasks at once
![alt text](https://user-images.githubusercontent.com/127349077/226676813-57259d8e-56b1-4d03-91cc-5dad60c4e139.jpg)

## 10. As a user, I want the RT5000 to be able to pause and resume tasks, so that I can make adjustments or take breaks during a task
![alt text](https://user-images.githubusercontent.com/127349077/226676823-1350bf82-f4f8-41ce-9b80-4c6dceb9c549.jpg)
